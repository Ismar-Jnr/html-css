Algumas tags, são tags antigas, que vem de versões anteriores do HTML. Boa parte delas são tags boas e funcionais até hoje, porém, existem algumas tags se tornar obsoletas com o passar do tempo.
Apesar disso, essas tags ainda podem funcionar por um certo período de tempo, mas não é aconselhavél usar essas tags, pois a qualquer momento elas podem ser desabilitadas.
Para checar alguma tag, se elas está obsoleta ou não, é recomendado que pesquise no site da w3c, que é a portadora oficial do html.
www.w3.org
